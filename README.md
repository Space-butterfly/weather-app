# Weather Detection Web App

A live weather detection web application built using HTML, CSS, and JavaScript.

## Features
- Real-time weather data using Open-Meteo API
- City-based weather search
- Temperature, humidity, wind speed, and conditions
- Notes saved using localStorage
- Modern glassmorphism UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Open-Meteo API
